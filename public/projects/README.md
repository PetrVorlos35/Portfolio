# Project screenshots

Drop a screenshot here for each project to light up the hover preview
(desktop, cursor-follow) and the inline thumbnail (mobile). Recommended:
~1200×900 (4:3), PNG or JPG, optimized.

Expected filenames (referenced from `locales/cs.ts` + `locales/en.ts`):

- `journeo.png`      — Journeo
- `outlay.png`       — Outlay
- `portfolio.png`    — Minimalist Portfolio
- `budget.png`       — Budgeting app
- `journeo-old.png`  — Journeo old
- `nba.png`          — NBA Guessing Game

If a file is missing, the preview/thumbnail for that project is hidden
automatically — nothing breaks and no placeholder is shown.
